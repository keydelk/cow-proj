# Cow-Proj - the Cow Project
**A silly little project to learn how to use LLMs via an API**

## Why use the Cow-Proj?
If you are curious and want to play around with it. It doesn't
do anything useful, just something for a little entertainment.

I'm using this personally to practice coding and working with the
Google Gemini API. If you want to play with this, you will need
your own key. Fortunately, you can get a key on the free-tier of
Google AI/Google One.

You will need a Google account, and set up an API key on the
[Google AI Studio](https://aistudio.google.com/).
